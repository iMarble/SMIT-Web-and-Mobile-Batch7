Topics : Lists & Table
Tags   : div, span, a, audio, and video

Lists : 
	There are 3 type of list in html that are as 
	follows :
	1- Order list
	2- UnOrder list
	3- Defination list

Table : 
	Data will be tabular format. We generally used rows & cols
        in it.
Div tag : 
        - This tag is just used to divied the portion of web page so
        we can style it accordingly.
        - This is Block level tag.
Span tag: 
        - This is just inline element that used to wrap around a text
        we can style it accordingly.
